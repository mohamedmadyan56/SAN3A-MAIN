🗃️ 1. تحديث Prisma Schema
تم التعديل prisma/schema.prisma
إضافة حالات طلب جديدة إلى enum RequestStatus:
  • SELECTED - تم اختيار الحرفي من قبل العميل
  • ACCEPTED - قبل الحرفي الطلب
  • ARRIVED - وصل الحرفي إلى الموقع
  • DISPUTED - يوجد نزاع على الطلب
  • REFUNDED - تم استرداد المبلغ
إزالة مسار الإخراج المخصص للمكتبة (output: ../src/generated/prisma) - يستخدم الآن @prisma/client الافتراضي.
🔌 2. تحديث تهيئة Prisma Client
تم التعديل src/utils/prisma.js
تغيير مسار الاستيراد من المسار المخصص إلى المكتبة الرسمية:
// قبل: const { PrismaClient } = require('../generated/prisma');
// بعد: const { PrismaClient } = require('@prisma/client');
🔄 3. إعادة كتابة Admin Controller
تم التعديل src/controllers/adminController.js
تم تحويل جميع استعلامات Mongoose إلى Prisma:
  • User.countDocuments() → prisma.user.count()
  • Request.aggregate() → prisma.request.aggregate() أو groupBy()
  • Request.find().populate() → prisma.request.findMany({ include: {...} })
  • User.findByIdAndUpdate() → prisma.user.update()
  • User.findById() → prisma.user.findUnique()
  • البحث بـ $regex → { contains, mode: 'insensitive' }
  • $lookup في aggregate → استعلام Prisma منفصل مع findMany + in
  • Request.aggregate $dateToString → prisma.$queryRaw (SQL خام)
  • request.save() → prisma.request.update()
📊 4. إعادة كتابة Dashboard Controller
تم التعديل src/controllers/dashboardController.js
تحويل جميع استعلامات Mongoose إلى Prisma:
  • Request.find({ client: userId }) → prisma.request.findMany({ where: { clientId: userId } })
  • $nin: ['COMPLETED', ...] → status: { notIn: [...] }
  • $in: ['SELECTED', ...] → status: { in: [...] }
  • $sum: '$pricing.totalAmount' → _sum: { totalAmount: true }
  • req.user._id → req.user.id
🚏 5. تحديث User Routes
تم التعديل src/routes/userRoutes.js
تحويل الـ inline handlers التي كانت تستخدم Mongoose مباشرة:
  • GET /public/:id - User.findById() → prisma.user.findUnique()
  • GET /craftsmen - User.find() → prisma.user.findMany()
  • PATCH /update-profile - User.findByIdAndUpdate() → prisma.user.update()
  • تحديث الحقول: location.address → locationAddress (يتوافق مع Prisma schema)
  • تغيير req.user._id → req.user.id
