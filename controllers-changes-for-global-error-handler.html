📋 التعديلات المطلوبة في الـ Controllers

الفكرة: كل catch blocks حالياً بتعمل res.status().json() مباشر. عشان الـ global error handler يشتغل، نحولها لـ next(err).
خطوة إضافية (اختياري): لو عاوز تخلي الـ validation errors (early returns) زي return res.status(404).json(...) تعدي كمان على الـ global handler، اعمل class صغير اسمه AppError: 

// backend/src/utils/appError.js
class AppError extends Error {
  constructor(message, statusCode) {
    super(message);
    this.statusCode = statusCode;
    this.status = String(statusCode).startsWith('4') ? 'fail' : 'error';
    this.isOperational = true;
    Error.captureStackTrace(this, this.constructor);
  }
}
module.exports = AppError;
